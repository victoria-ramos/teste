# TESTE TÉCNICO - ASIMOV

## FERRAMENTAS UTILIZADAS:
- Figma MCP, Dembramdt, Claude Code, Claude Design e Gemini. 

## ETAPAS FACILITADAS PELA IA:
- Brainstorm de solução/design, organização de referências, handoff do design system e extração de identidade visual, código inicial e code-review.

## ETAPAS MANUAIS:
- Revisão do código inicial, ajustes finos de css/html, separação em componentes, aplicação de clean-code. Por mais que a IA ajude no bruto, acredito que o senso crítico humano é fundamental, saber e entender os conceitos aplicados, aspectos de segurança, avaliar se o código está de fácil compreensão e de fácil manutenção são fatores importantes a se observar, principalmente em projetos grandes e às vezes a IA pode passar batido nesses temas.
